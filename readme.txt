作品1、作品2はどちらも MATLABというプログラミング言語から作成しました。
いずれの作品も実行ファイルは MATLAB Runtimeが無ければ動作しません。
実行するには、"\MATLAB_Runtime_R2022a_Update_6_win64\setup.exe"から MATLAB Runtimeをインストールしてください。
詳細はhttps://jp.mathworks.com/help/compiler/install-the-matlab-runtime.htmlにあります。

作品1は大学生の時に卒業研究で実際に作成したシミュレーションプログラムになります。
作品2はその卒業研究の課程で作成したシミュレーションプログラムです。
